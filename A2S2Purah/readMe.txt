Welcome to group Purahs XML Parser program.


How to run this program:

	1.  Export the entire zipped folder to a destination of you choice.


	2.  Navigate inside the folder "A2S2Purah".


	3.  Right click inside this folder and select the option: "Open in Terminal".
	    If you don't see this option try: shift + right click.


	4.  In the command line, enter the following command (do not include curly brackets):


	    java -jar Parser.jar {file_path}


	    example:
	    java -jar Parser.jar C:\temp\MyXmlFile.xml
